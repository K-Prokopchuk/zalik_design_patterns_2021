У нас є гра
Для відображення використовуємо Singleton класс Screen 
Також є будівельник карт для гри та абстрактна фабрика створення мобів та класс вектору реалізуючий шаблон Прототип

Будівельник карт потрібен для легкого та кастомного білду карт з різними типами противників
Абстрактна фабрика мобів найкраще усього підходить для створення противників зі спеціальними настройками
Векторів у іграх багато і їх часто потрібно клонувати, тому шаблон прототип ідеально підходить
Оскільки герой у нас тільки один і він в один момент може бути тільки на одній карті, то шаблон одинак для його екрану - ідеальний варіант
